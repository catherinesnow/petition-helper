Instructions

  1. Copy and paste rows from the submitted petitions spreadsheet.
  2. Click on a petition image to expand it and to set it as active to use the Change Request dropdown.
  3. While the image is expanded, you can use the dropdown to add a change request to the row.
  4. You can manually enter text without selecting the image, if you'd like.
  5. Click on the image again to deselect it.
  6. When you are done entering changes, click the "Create Message" to generate a table to copy and paste into the email template.
  7. You can also copy the name and email.
